# Suirobo
2025水中ロボットコンテスト用の制御プログラムです。

実行するにはESP32のライブラリのインストールと各種コントローラー用のライブラリを導入する必要があります。

ESP32はファイル/基本設定から設定画面を開き追加のボードマネージャのURLに

`https://dl.espressif.com/dl/package_esp32_index.json`を追加してください
